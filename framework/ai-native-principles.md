# AI Native Principles

- The layout no longer displays everything all the time. Components are surfaced contextually, serving the right content for the current action at the current process stage, not a static page template.
- Al agents pre-investigate the case before the agent arrives, verifying order status, checking policies, and scoring risk, so agents make judgment calls, not research calls.
- In Conversation-First mode, components are generated dynamically in the moment of need rather than pre-loaded onto the page.
- Ul-Driven mode is not a legacy fallback, it is a first-class Al Native experience where Al-assembled components and inline findings support familiar agent navigation patterns.
- Dynamic components self-manage their own content, updating independently as Al agents query new data or workflow rules trigger changes, without requiring agent action or page refresh.
- Human autonomy is preserved across all three modes. Al pre-investigates and surfaces suggested next steps; the agent always owns the final Approve or Deny decision.
